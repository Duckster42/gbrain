# DuckMemory Clean-Room Spec Inputs
- Operation contract model: typed command/query definitions, scope labels, transport adapters.
- Data model blueprint: pages/chunks/links/tags/facts/takes/files/sources/jobs/audit.
- Retrieval blueprint: lexical/vector fusion + stage attribution schema.
- Graph blueprint: provenance edges + traversal constraints + ranking hooks.
- Enrichment DAG: extract → validate → reconcile → consolidate → score.
- Security blueprint: explicit trust context, capability tokens, source-scoped ACL.
- Eval blueprint: replay corpus, pairwise significance, golden query suites.
- Runtime blueprint: minimal dependencies, offline-first local mode, optional managed backend.
